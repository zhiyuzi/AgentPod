name: sysinfo
description: 收集并展示当前系统信息（OS、CPU、内存、磁盘、Python 版本）。当用户询问系统状态、环境信息或运行环境时使用。
---

收集当前系统的运行环境信息并以结构化方式展示给用户。

## 步骤

1. 执行 `scripts/collect.sh` 收集系统信息
2. 将输出整理为清晰的表格或列表格式
3. 如果某项信息获取失败，标注为"未知"而非报错

## 输出格式

```
系统信息
├── OS: ...
├── 内核: ...
├── CPU: ...
├── 内存: 已用/总量
├── 磁盘: 已用/总量
└── Python: ...
```
